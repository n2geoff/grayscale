Getting Started

Grayscale CSS provides a minimalistic approach to styling by putting layout in the forefront, removing color, allowing for developers to rapidly prototype layouts. Everything is extendable, so just apply your theme later! All in, the package is only 2kb gzipped!

Buttons

Button styles are automaticly applyed for inherited button types and you can add the .button class to links. In addition, you can mix the shades above for variety.

Grid

Grayscale CSS uses a precent-based fluid grid system with 6 column sizes, and a max width of 1140px. Add a column-% class to every column, where % can be 20, 25, 33, 50, 100. For example:
