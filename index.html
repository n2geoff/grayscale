Utility

Below are some helper and modifer classes. All modifier classes start with .is-
Style Description Style Description
.is-well Adds .5em padding to tag .is-indented adds a left margin
.is-wall Adds .5em padding-left and padding-right to tag .is-borderless removes all borders
.is-wide sets tag width 100% .is-hidden sets display to none
.is-tall sets tag height 100% .is-upper sets font to uppercase
.is-fit sets tag width / height 100% .is-lower sets font to lowercase
.is-right float tag right .is-title capitalizes font
.is-left float tag left .is-muted lightens font color
.is-centered centers tag .is-block applys display:block
.is-inline applys display:inline .is-clear applys clear:both
.is-small decreses font size .is-large increase font size
.is-divider draws hr rule
