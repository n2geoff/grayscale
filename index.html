Getting Started

Grayscale CSS provides a minimalistic approach to styling by putting layout in the forefront, removing color, allowing for developers to rapidly prototype layouts. Everything is extendable, so just apply your theme later! All-in, the package is only 2kb gzipped!

Buttons

Button styles are automaticly applyed for inherited button types and you can add the .button class to links. In addition, you can mix the shades above for variety.

Extend

It is easy to extend buttons with your own css, for example

Add Color

.is-primary {color: #FFF; background-color: blue;}

Add Flair

.is-rounded {border-radius: .5px;}

Grid

Grayscale CSS uses a precent-based fluid grid system with 6 column sizes, and a max width of 1140px. Add a .column-% class to every column, where % can be 20, 25, 33, 50, 100. For example:

Utility

Below are some helper and modifer classes. All modifier classes start with .is-
Style Description
.is-well Adds .5em padding to tag
.is-wall Adds .5em padding-left and padding-right to tag
.is-wide sets tag width 100%
.is-tall sets tag height 100%
.is-fit sets tag width / height 100%
.is-right float tag right
.is-left float tag left
.is-centered centers tag
.is-block applys display:block
.is-inline applys display:inline
.is-clear applys clear:both
