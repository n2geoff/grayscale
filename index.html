Shades of Gray

The core of Grayscale CSS revolves arounds shades of gray to help quickly block out your designs, as such it provides helper classes to get you a full range of shades from white to black, however A few colors were added for splash, but these can easly be overridden
.is-white
.is-light
.is-gray
.is-dark
.is-black

Buttons

Button styles are automaticly applyed for inherited button types and you can add the .button class to links. In addition, you can mix the shades above for variety.

Page Layout

Grayscale CSS Layout system is flexbox-based, for a max width of 1140px. Using .row and .col creates a dynamic layout that can scale by a -# magintude like .col-2 for 2x the size of col-1 This magitude # can be anywhere from 1-5 .col-1, .col-2, .col-3, .col-4, .col-5.

Utility

Below are some helper and modifer classes. All modifier classes start with .is-
Style Description
well Adds .5rem padding to tag
.wall Adds .5em padding-left and padding-right to tag
.wide sets tag width 100%
.tall sets tag height 100%
.fit sets tag width / height 100%
.right float tag right
.left float tag left
.centered centers tag
.inline applys display:inline
.small decreses font size
.divider draws hr rule
Style Description
.indented adds a left margin
.borderless removes all borders
.hidden sets display to none
.uppercase sets font to uppercase
.lowercase sets font to lowercase
.title capitalizes font
.muted lightens font color
.block applys display:block
.clear applys clear:both
.large increase font size
